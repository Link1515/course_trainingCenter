標籤(元素)使用注意哪些?

  • 標籤雖不分大小寫，養成習慣使用小寫
  • 會有< >開始，</ >結束，起始標籤與結束標籤，裡面為標記內容
  • 標籤具有屬性與值的特性，屬性之間需要空格
  • 屬性值雖然埠限制單引號或雙引號，養成習慣使用雙引號

標籤(元素)的構成

  • id 與 class 是全域屬性，每個標籤都可以擁有這兩個屬性，他們很重要，常被使用在CSS、JS中
  • id 屬性，每份html文件只能擁有一個 id ，不能有重複的名稱
  • class 屬性，可以擁有一個或多個類別名稱的值，樣式專用的屬性
