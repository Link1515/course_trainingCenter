position: relative

當元素為相對定位時...

  • 使用位移屬性會以自己原來的位置位移，並保留本身員來的空間，relative 會說進入定位層
  • 沒有任何位移屬性，會留在原地不動。這種relative元素通常做為absolute 子元素位移目標的依據

position: absolute

  • 當沒有設定位移屬性，改元素會脫離正常流向，升到定位層
  • 若元素都找不到 absolute、relative、fixed，最後會以最初容器根元素(html)作為位移目標

絕對定位與float

  • 兩者都會脫離父元素的正常流獨立出來，絕對定位會在定位層，浮動在浮動層
  • 兩者都是BFC成員，都會讓元素變成區塊級(block)
  • 另外，position: fixed 也會是區塊級元素，也是BFC一員
