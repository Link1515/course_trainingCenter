id 選擇器

  • 一份文件裡面，只允許一個獨一無二的 id 名稱(就像身分證)，通常作為網頁邏輯區塊的名稱
  • 對程式而言(JavaScript、jQuery)，id必須唯一，如果有兩個相同id，只會選擇到第一次遇到的id最為目標。id 選擇器最簡單、最快速、最直接的選擇方式
