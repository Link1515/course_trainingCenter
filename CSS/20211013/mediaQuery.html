響應式設計(Responsive web design, RWD)

  • 1. 響應式網頁設計，內容如流水般能適應各種形狀的容器，能在每個裝置尺寸正常瀏覽
  • 2. CSS 媒體查詢(Media Query)負責最主要的工作
  • 3. 媒體查詢的斷點(breakpoint)位不同裝置尺寸設定通過的位置點，當通過設定的位置點套用該尺寸的樣式
  • 4. 行動裝置的漸進增強的概念，基於小尺寸開始思考內容的設計模式
