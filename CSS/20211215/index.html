01 Sass 介紹

Sass 是 CSS 的預處理器，目的在減少CSS的重工，以及更有效率維護樣式結構，他須進行編譯

Sass 可以使用變數、函數、流程控制、嵌套、繼承、混入等功能

02. Sass 的資料型態

  1. 1. number: 數字有分有單位與無單位
  2. 2. String: 字串分有無引號
  3. 3. color: blue, #04a3f9
  4. 4. boolean: ture / false
  5. 5. null
  6. 6. list(像陣列): 用半形空格或逗號做為區隔
  7. 7. map(像物件): 類似 JS 物件

03. 變數

宣告使用 $ ，在大括號裡面的為區域變數，寫在最外面為全域變數

變數的 _ 與 - 可以互換使用，如 $bg-dark 與 $bg_dark 是一樣的變數

!default 是預設，若變數前後已經被賦值，會引用套用的值，!default 就不會被套用

使用變數可以維持網站一致性，節省修改成本
